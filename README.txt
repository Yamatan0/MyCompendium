ОПИСАНИЕ
Данный модуль представляет из себя набор компендиумов с переведенными классами, классовыми и 
расовыми способностями, чертами и предметами из основных книг по D&D 5e (PHB, XGE, SCG и другие).

Огромная благодарность пользователям: Tiki, Лапть, ELFA, Vita, Енисей, zubanzub, Salamander, Riverander, Gecko за помощь в 
составлении этого модуля.

После этого на экране компендиумов должны появится следующие компендиумы:
"Враги Собственныe (RU)" - все листы, которые используются для призванных НПЦ.
"Заклинания (RU)" - полный перечень доступных заклинаний из основных книг;
"Классовые Способности (RU)" - полный перечень способностей классов из основных книг;
"Классы Персонажей (RU)" - полный перечень классов из основных книг;
"Расовые Способности (RU)" - перечень некоторых частоиспользуемых расовых способностей из основных книг;
"Черты (RU)" - перечень черт из PHB,XGE и TCoE;
"Предметы (RU)" - полный перечень предметов, как обычных так и магических, из основных книг.
